慣用句

Kotlinでよく使用される慣用句を集めた。もし好みの慣用句があるならプルリクエストを投げて貢献して欲しい。

DTOを作成する (POJO/POCO)

data class Customer(val name: String, val email: String)

Customerクラスは次の機能を提供する：

  • 全てのプロパティに対するゲッター (そしてvarキーワードが使用されたときはセッターも )
  • equals()
  • hashCode()
  • toString()
  • copy()
  • すべてのプロパティに対して、component1(), component2(), …, (Data classesを参照のこと。)

関数の引数に対するデフォルト値

fun foo(a: Int = 0, b: String = "") { ... }

リストのフィルタ

val positives = list.filter { x -> x > 0 }

または、こう短くも書ける：

val positives = list.filter { it > 0 }

Stringの補完

println("Name $name")

インスタンスのチェック

when (x) {
    is Foo -> ...
    is Bar -> ...
    else   -> ...
}

mapやlistのペアを書き下す

for ((k, v) in map) {
    println("$k -> $v")
}

k, vの名前は特に定まっていない。

範囲の使用

for (i in 1..100) { ... }
for (x in 2..10) { ... }

読み取り専用のlist

val list = listOf("a", "b", "c")

読み取り専用のmap

val map = mapOf("a" to 1, "b" to 2, "c" to 3)

mapにアクセスする

println(map["key"])
map["key"] = value

遅延評価プロパティ

val p: String by lazy {
    // 文字列を計算する
}

拡張関数

fun String.spaceToCamelCase() { ... }

"Convert this to camelcase".spaceToCamelCase()

シングルトンの作成

object Resource {
    val name = "Name"
}

if not nullの速記法

val files = File("Test").listFiles()

println(files?.size)

if not null and elseの速記法

val files = File("Test").listFiles()

println(files?.size ?: "empty")

if null文の実行

val data = ...
val email = data["email"] ?: throw IllegalStateException("Email is missing!")

if not nullの実行

val data = ...

data?.let {
    ... // nullでなければこのブロックが実行される
}

when文でreturnする

fun transform(color: String): Int {
    return when (color) {
        "Red" -> 0
        "Green" -> 1
        "Blue" -> 2
        else -> throw IllegalArgumentException("Invalid color param value")
    }
}

‘try/catch’式

fun test() {
    val result = try {
        count()
    } catch (e: ArithmeticException) {
        throw IllegalStateException(e)
    }

    // resultを使用する
}

‘if’式

fun foo(param: Int) {
    val result = if (param == 1) {
        "one"
    } else if (param == 2) {
        "two"
    } else {
        "three"
    }
}

Unitを返すビルダースタイルの使用

fun arrayOfMinusOnes(size: Int) {
    return IntArray(size).apply { fill(-1) }
}

単一行関数

fun theAnswer() = 42

これは次と同等である：

fun theAnswer(): Int {
    return 42
}

これは他の慣用句と組み合わせることができ、コードを短くすることにつながる。例） when-式:

fun transform(color: String): Int = when (color) {
    "Red" -> 0
    "Green" -> 1
    "Blue" -> 2
    else -> throw IllegalArgumentException("Invalid color param value")
}

あるオブジェクトのインスタンスで ‘with’ を使って複数の関数を呼ぶ

class Turtle {
    fun penDown()
    fun penUp()
    fun turn(degrees: Double)
    fun forward(pixels: Double)
}

val myTurtle = Turtle()
with(myTurtle) { // 100ピクセル四方の四角形を描く
    penDown()
    for(i in 1..4) {
        forward(100.0)
        turn(90.0)
    }
    penUp()
}

Java 7のtry-with-resources

val stream = Files.newInputStream(Paths.get("/some/file.txt"))
stream.buffered().reader().use { reader ->
    println(reader.readText())
}
